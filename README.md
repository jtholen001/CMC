TO ADD PROJECT TO ECLIPSE:
1. Open Eclipse
2. Go to "File" > "Import..."
3. Under "General", select "Existing Projects into Workspace"
4. Click "Next" button
5. Choose "Select archive file:" and click "Browse..." button
6. Open CMC-main.zip
7. Click "Finish" button
